# 项目简介

该项目是一个Python全国地铁出行数据分析系统，包含数据集和源代码。

## 核心功能

- 数据集：包含38个城市地铁出行数据，包括数据概况、分客流量、每日客流、客流量预测、历史客流top10、年日均客流、周日均客流、月日均客流等。
- 数据处理：使用ARIMA时间序列模型对客流量进行预测。
- 可视化：利用matplotlib、seaborn、plotly库生成散点图、折线图、旭日图、动态散点图和菜单小提琴图等可视化效果。

## 适用场景

- 城市地铁运营管理：为地铁运营提供客流数据分析和预测，优化运营策略。
- 地铁规划设计：为地铁线路规划和设计提供数据支持。
- 市场研究：为相关企业提供地铁出行市场分析。

## 技术特点

- 数据来源全面：包含全国38个城市地铁出行数据，覆盖范围广。
- 预测准确：采用ARIMA时间序列模型进行客流量预测，具有较高的准确性。
- 可视化效果丰富：多种可视化图表，直观展示数据特征。
